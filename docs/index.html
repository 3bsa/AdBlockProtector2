Installation instructions

AdBlock Protector 2 is a wrapper around uBlock Protector.

If you are using Chromium / Chrome and uBlock Origin, please check it out.


AdBlock Protector 2 is now in developer alpha. Not a developer? AAK-Cont, although unmaintained, will still be available for download.


Step 1. Enable Adblock Warning Removal List.

Step 2. Install AdBlock Protector 2 List (choose the adblock that you are using)

  • uBlock Origin

  • AdBlock or Adblock Plus (COMING SOON!)

READ THIS CAREFULLY: I am fully aware that the filter list for AdBlock and Adblock Plus is not yet available, do not open issues about this, I'm working on it.

Step 3. Install AdBlock Protector 2 Extension (choose the browser that you are using)

  • Chromium / Chrome / Opera (COMING SOON!)

  • Quantum (formerly Firefox) (COMING SOON!)

  • Edge (Planned)

  • Safari (Help Wanted)

WARNING: You must install the extension, as it includes a local server as a surrogate for real ads servers, installing only the filter list will cause requests to hit the real ads servers instead.

READ THIS CAREFULLY: For now, you need to load the unpacked extension in developer mode (clone the repository and load the extension folder into your browser), I will publish them to extension stores eventually, do not open issues about this.


Click on the extension icon if you need more information.
