AdBlock Protector 2

An anti-adblock defuser

Installation instructions

AdBlock Protector 2 is a wrapper around uBlock Protector, if you are using Chromium / Chrome and uBlock Origin, please check it out.


Step 1. Enable Adblock Warning Removal List in your adblocker settings page.

Step 2. Install the Click AdBlock Protector 2 List for your adblocker

  • uBlock Origin

  • AdBlock or Adblock Plus

Step 3. Install AdBlock Protector 2 Extension.


Click on the extension icon if you need more information.
