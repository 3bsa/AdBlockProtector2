Installation instructions

AdBlock Protector 2 is a wrapper around Nano Defender.

If you are using Chromium / Chrome and Nano Adblocker / uBlock Origin, please check it out.


AdBlock Protector 2 is now in developer alpha. Not a developer? AAK-Cont, although unmaintained, will still be available for download.


Step 1. Enable Adblock Warning Removal List.

Step 2. Install AdBlock Protector 2 List (choose your adblocker)

  • uBlock Origin (COMING SOON!)

  • AdBlock or Adblock Plus (COMING SOON!)

Step 3. Install AdBlock Protector 2 Extension (choose your browser)

  • Chromium / Chrome / Opera (COMING SOON!)

  • Quantum (formerly Firefox) (COMING SOON!)

  • Edge (Planned)

  • Safari (Help Wanted)


Click on the extension icon if you need more information.
