AdBlock Protector 2

An anti-adblock defuser

Installation instructions

AdBlock Protector 2 is a wrapper around uBlock Protector.

If you are using Chromium / Chrome and uBlock Origin, please check it out.


AdBlock Protector 2 is now in developer beta. Not a developer? AAK-Cont, although unmaintained, will still be available for download.


Step 1. Enable Adblock Warning Removal List in your adblocker settings page.

Step 2. Install AdBlock Protector 2 List (choose the adblock that you are using)

  • uBlock Origin

  • AdBlock or Adblock Plus (COMING SOON!)

READ THIS CAREFULLY: I am fully aware that this project does not work with AdBlock nor Adblock Plus for now, do not open issues about this, I'm working on it.

Step 3. Install AdBlock Protector 2 Extension (choose the browser that you are using)

  • Chromium / Chrome / Opera (COMING SOON!)

  • Quantum (formerly Firefox) (COMING SOON!)

  • Edge (Planned)

READ THIS CAREFULLY: For now, you need to load the unpacked extension in developer mode, I will publish them to extension store eventually, do not open issues about this.


Click on the extension icon if you need more information.
